Knapp

Ein knappevariant. Kvit bakgrunn, accent-farge når aktiv.

Knapp-detaljar

Bruk .btn-klassen. Hover-effekt løftar knappen opp (transform) og gir den accent-farge og redusert skugge.

Responsiv standard (2026)

Denne malen brukar flytande typografi og "intrinsic design".

Fluid Typography

Overskrifter og tekst brukar clamp() og skalerer automatisk mellom desktop og mobil utan faste "breakpoints".

Flex & Grid

Oppsett og knapperader brukar flex-wrap: wrap; og minmax() slik at innhald bryt linje når plassen blir for smal.

Fluid Spacing

Sidan rammer og skuggar er tjukke, brukar padding og margin òg clamp() for å knipe strammare på smale mobilar.

Modals

Fem modal-typar som speglar dei fem bokstypane. modal4 og modal5 er dobbelt så brede for større innhold. Klikk for å opne:

Modal-oversikt

modal1: Accent shadow — same stil som box1.

modal2: Tilta (-1deg) med teipbit — same stil som box3.

modal3: Header + body + footer — same stil som box4.

modal4: Avrunda hjørner — same stil som box2. Dobbel bredde.

modal5: Fane-navigasjon + body — same stil som box5. Dobbel bredde.
